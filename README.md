# CrossHermit Map Editor

A custom map editor designed for editing and creating levels/maps.

## Live Demo & Usage
* **Use it live:** [CrossHermit Map Editor Live](https://freeze-man-1.github.io/CrossHermitMapEditor/)
* **Run locally:** You can clone this repository and run it locally on your machine.

---

## Features
Based on the current source implementation, the editor supports the following features:
* **Map Rendering & Visualization:** View and edit map grids, layers, and tiles dynamically.
* **Layer Management:** Supports editing multiple map layers (such as collision, background, and foreground elements).
* **Tile & Collision Editing:** Place tilesets and modify collision maps directly within the interface.
* **Import & Export:** Easily load and save map configurations in JSON format to integrate with your game projects.


---

## Roadmap & In-Development
* **Re-importing Map Data:** We are actively working on improving the workflow for re-importing and updating existing map data seamlessly.
* **New Map Support:** Expanding capabilities to create and support entirely new map structures and layouts.
